# 横滨设施预约助手 (Yokohama Facility Booking Assistant)

版本：**v1.1**
最后更新：**2026-03-04**

Chrome / Edge 扩展程序，用于在横滨市市民利用设施预约系统中自动执行空位搜索流程（本地运行、无外部服务器）。

## v1.1 更新内容

- 搜索页支持记忆“上次点击搜索”的条件，并在下次打开弹窗时自动回填。
- 方案页支持“编辑已保存方案”，可回填到搜索页后覆盖更新。
- 清理未使用代码与冗余样式，提升可维护性。
- 优化空结果页识别逻辑，降低空位误判风险。
- 文档与权限说明同步到最新实现。

## 主要功能

- 一键后台搜索：自动填写条件并执行查询。
- 方案保存/运行：保存常用条件，后续一键运行。
- 方案编辑：在方案页回填并更新现有方案。
- 定时检查：按 15/30/60 分钟周期自动运行启用方案。
- 中日双语界面：支持中文/日文切换。
- 桌面通知：发现空位时发送系统通知。

## 安装方式

1. 打开 `chrome://extensions/`（Edge 用 `edge://extensions/`）。
2. 开启“开发者模式”。
3. 选择“加载已解压的扩展程序”。
4. 选择项目目录 `Badminton-YoYaku`。

## 使用流程

1. 首次使用先点“同步”，获取最新利用目的/区域字典。
2. 在搜索页设置条件：
   - 利用目的（可多选）
   - 区域（可多选）
   - 搜索期间（相对今天：7天/14天/21天/1个月）
   - 时间段
   - 星期过滤（含祝日）
3. 点击“🚀 一键搜索（后台运行）”。
4. 若常用条件，点击“保存为方案”。
5. 在“方案”页可执行：
   - `▶` 直接运行
   - `✎` 编辑并更新
   - `✕` 删除
   - `⏱️` 开关定时检查

## 权限说明

- `storage`：保存本地设置、方案、字典、最近搜索条件。
- `scripting`：在目标站点注入采集脚本（字典同步）。
- `alarms`：执行定时检查。
- `notifications`：推送空位通知。
- `host_permissions`：仅限 `https://www.shisetsu.city.yokohama.lg.jp/*`。

## 维护说明

- 本项目依赖目标网站 DOM；若官方改版，需要同步更新选择器。
- 所有数据仅存储于 `chrome.storage.local`，不会上传到第三方服务器。
