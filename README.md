# 横浜施設予約アシスタント

> 面向横滨市市民利用设施预约系统的 Chrome 扩展。用于减少重复检索操作，支持参数同步、后台搜索、方案保存、定时检查、空位通知，以及中文 / 日文界面切换。

- 当前版本：`v1.1.0`
- 最后更新：`2026-03-24`
- 适用站点：[横滨市市民利用设施预约系统](https://www.shisetsu.city.yokohama.lg.jp/)
- 扩展地址：[Chrome Web Store - 横浜施設予約アシスタント](https://chromewebstore.google.com/detail/%E6%A8%AA%E6%B5%9C%E6%96%BD%E8%A8%AD%E4%BA%88%E7%B4%84%E3%82%A2%E3%82%B7%E3%82%B9%E3%82%BF%E3%83%B3%E3%83%88/knhnnfllghnlgajgmahpokfghlnlbdod)

## 项目简介

在横滨市设施预约网站上反复切换条件、刷新日期和检查空位会比较耗时。这个扩展把高频重复动作整理成浏览器内的本地自动化流程：

- 首次同步网站上的利用目的和区域参数
- 根据设定条件自动打开页面并执行搜索
- 保存常用条件为方案，后续一键复用
- 定时检查已启用的方案，并在发现空位时发送系统通知

整个流程不依赖外部服务器，数据仅保存在浏览器本地存储中。

## 核心功能

- 参数字典同步：从目标网站抓取最新的利用目的和区域选项，避免手动维护枚举值。
- 后台一键搜索：自动填写搜索条件并执行查询流程，减少重复点选。
- 方案保存与编辑：支持保存常用条件、重新编辑已有方案、直接运行或删除方案。
- 记忆最近一次搜索：重新打开弹窗时自动回填上次实际执行搜索时的条件。
- 定时检查：按 `15 / 30 / 60` 分钟周期运行已启用的方案。
- 空位高亮与通知：发现空位后发送系统通知，并在结果页面高亮可用时段。
- 中日双语界面：支持中文和日文切换。
- 本地保存设置：可在设置页保存登录 ID、密码和检查间隔。

## 安装方式

### 方式一：从 Chrome Web Store 安装

1. 打开 [Chrome Web Store - 横浜施設予約アシスタント](https://chromewebstore.google.com/detail/%E6%A8%AA%E6%B5%9C%E6%96%BD%E8%A8%AD%E4%BA%88%E7%B4%84%E3%82%A2%E3%82%B7%E3%82%B9%E3%82%BF%E3%83%B3%E3%83%88/knhnnfllghnlgajgmahpokfghlnlbdod)。
2. 点击安装扩展。
3. 安装后将扩展固定到工具栏，方便后续使用。

### 方式二：开发者模式加载

1. 下载或克隆本项目。
2. 打开 `chrome://extensions/`，如使用 Edge 则打开 `edge://extensions/`。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择当前仓库根目录 `badminton-yoyaku`。

## 使用流程

1. 首次使用先点击“同步”，从网站获取最新的利用目的和区域字典。
2. 在搜索页设置条件：
   - 利用目的
   - 区域
   - 搜索期间
   - 时间段
   - 星期过滤和祝日过滤
3. 如有需要，在设置页填写登录 ID、密码，以及定时检查间隔。
4. 点击“后台一键搜索”开始执行。
5. 如果这组条件会重复使用，可以保存为方案。
6. 在“方案”页中可直接执行、编辑、删除方案，或开启该方案的定时检查。
7. 当检测到空位时，扩展会发送桌面通知，并把结果页保留给你继续查看。

## 权限与数据说明

- `storage`：保存语言、设置、方案、参数字典、最近一次搜索条件等本地数据。
- `scripting`：在目标站点注入脚本，用于采集参数字典。
- `alarms`：按设定周期执行定时检查。
- `notifications`：在发现空位时发送系统通知。
- `host_permissions`：仅访问 `https://www.shisetsu.city.yokohama.lg.jp/*`。

补充说明：

- 所有数据保存在 `chrome.storage.local`。
- 不依赖第三方后端服务。
- 登录信息如有填写，也仅保存在当前浏览器本地。
