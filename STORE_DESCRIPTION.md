# 应用商店上架简介（v1.1）

> 用于 Chrome Web Store 的“简介摘要”和“详细说明”文案。
> 最后更新：2026-03-04

## 简介摘要 (Short Description)

横滨市设施预约自动化助手。支持后台一键搜索、空位通知、方案保存与编辑、定时检查。纯本地运行，不上传个人数据。

## 详细说明 (Detailed Description)

在横滨市预约体育设施时，重复勾选条件、反复刷新空位非常耗时。
**横浜施設予約アシスタント v1.1** 帮你把这些步骤自动化。

### 核心功能

- 🚀 后台一键搜索：自动填写条件并执行搜索流程。
- 🎉 空位实时通知：发现空位后立即弹出系统通知。
- 💾 方案保存与复用：常用条件可保存为方案，一键执行。
- ✏️ 方案编辑（v1.1）：可直接编辑并更新已保存方案。
- 🧠 记忆上次搜索（v1.1）：自动回填最近一次点击搜索时的条件。
- ⏱️ 定时自动检查：支持每 15/30/60 分钟后台执行。
- 🌐 中日双语：中文 / 日文界面切换。

### 隐私与安全

- 不采集、不上传用户数据。
- 不依赖第三方后台服务。
- 登录信息（如用户填写）仅保存在本机浏览器本地存储中。
